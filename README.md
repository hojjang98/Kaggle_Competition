이 프로젝트는 **[2024 SCU 혁신 AI 경진대회]**에 참가하여 데이터를 분석하고 분류 모델을 구축하는 과정을 다룹니다. 목표는 **데이터 전처리**와 결론적으로는 **갑상선 질환 이진분류** 였습니다.

### 🔍 문제 정의
